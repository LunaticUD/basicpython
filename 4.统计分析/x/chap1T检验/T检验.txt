# T检验是一种用于比较两个样本均值差异的方法，适用于小样本且总体标准差未知的情况。
# 它基于t分布理论，通过计算t统计量并将其与临界值对比来判断两组数据的均值是否存在显著性差异1。
# 单样本T检验：评估单一组样本的均值是否与已知总体均值有显著差异。
# 独立样本T检验：比较两个独立样本的均值是否有显著差异。
# 配对样本T检验：比较同一对象在不同条件下的测量结果之间的差异。
# TODO：公式推导过程
# 对于独立样本T检验，其公式为：
# 𝑡=(X拔1-X拔2)/根号下(s1方/n1+s2方/n2)
# 其中，X拔1,X拔2分别为两个样本的均值，𝑠1,𝑠2分别为两个样本的标准差，𝑛1,𝑛2分别为两个样本的大小。自由度𝑑𝑓=𝑛1+𝑛2−2。
